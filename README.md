
# ssproject

Technical test for SS


## Run Locally

Clone the project

```bash
  git clone https://github.com/ksanmugam/ssproject
```

Go to the project directory

```bash
  cd ssproject
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run serve
```
